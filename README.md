# PinkNotice
hiển thị giao diện thông báo overlay (chèn lên màn hình) cho game trên iOS, bạn cần sử dụng môi trường phát triển Theos. Dự án này sẽ hook (can thiệp) vào quá trình khởi động của ứng dụng và vẽ một UIView tùy chỉnh với màu hồng phấn như bạn yêu cầu. Dưới đây là toàn bộ cấu trúc dự án và mã nguồn để bạn có thể biên dịch (make) ra file .dylib.
